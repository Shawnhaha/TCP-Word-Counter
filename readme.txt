TCP socket
A word count server that can do the following: converts first alphabet of every word to upper-case alphabet, counts the number of words, counts the number of alphabets, and finally counts the number of vowels in a given sentence.

Put all files to one directory on cse machine.
Type "make" to run makefile.
After makefile runs, server and client will be compiled.
The executable files called "server" and "client".
Use ./server <Port Number> and ./client <Port Number> to run them.
Please run server and client with a same valid port number.
After using the program, you can use make clean to remove all executable files.
